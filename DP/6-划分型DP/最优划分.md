计算最少（最多）可以划分出多少段、最优划分得分等。
一般定义 `𝑓[𝑖]` 表示长为 𝑖 的前缀 `𝑎[:𝑖]`在题目约束下，分割出的最少（最多）子数组个数（或者定义成分割方案数）
方案：枚举最后一个子数组的左端点L, 从`f[L]` 转移到 `f[i]`, 并且考虑`a[L:i]`对最优解的影响

#  [132. 分割回文串 II](https://leetcode.cn/problems/palindrome-partitioning-ii/)
```python
class Solution:
    def minCut(self, s: str) -> int:
        n = len(s)
        def is_palindrome(s):
            return s == s[::-1]

        dp = [n] * n 
        # dp[i] 表示前i个字符的最小分割次数

        for i in range(n):
            if is_palindrome(s[:i+1]):
                dp[i] = 0
                continue

            for j in range(i):
                if is_palindrome(s[j+1:i+1]):
                    dp[i] = min(dp[i], dp[j] + 1)
        return dp[-1]
```

class Solution:
    def minCut(self, s: str) -> int:
        n = len(s)
        @lru_cache
        def is_palindrome(l, r): # inclusive
            if l >= r:
                return True
            return s[l] == s[r] and is_palindrome(l+1, r-1) 

        @lru_cache
        def dfs(i):
            if is_palindrome(0, i):
                return 0
            res = float('inf')
            for j in range(1, i+1):
                if is_palindrome(j, i):
                    res = min(res, dfs(j-1)+1)
            return res

        return dfs(n-1)

# [3578. 统计极差最大为 K 的分割方式数](https://leetcode.cn/problems/count-partitions-with-max-min-difference-at-most-k/)
给你一个整数数组 `nums` 和一个整数 `k`。你的任务是将 `nums` 分割成一个或多个 **非空** 的连续子段，使得每个子段的 **最大值** 与 **最小值** 之间的差值 **不超过** `k`。

返回在此条件下将 `nums` 分割的总方法数。

由于答案可能非常大，返回结果需要对 `109 + 7` 取余数。

```python
class Solution:
    def countPartitions(self, nums: List[int], k: int) -> int:
        MOD = 10**9 + 7
        n = len(nums)
        dp = [0] * (n + 1) # dp[i]表示前i个数能组成的合法子数组个数
        # dp[i] = sum(dp[j] for j in range(i) if nums[j..i-1]的差值 ≤ k)
        # 所以某种意义上我们希望对于每个i找到最小的j，使得nums[j..i-1]的差值 ≤ k；那么这样情况下累加的项目就会更多
        prefix = [0] * (n + 1) # dp的前缀和
        dp[0] = 1 # 边界条件（没有数，唯一一个合法子数组（空数组））
        prefix[0] = 1
        
        max_q = deque()  # 存下标，维护递减
        min_q = deque()  # 存下标，维护递增
        left = 0
        
        for i in range(1, n + 1):
            val = nums[i - 1]
            # 维护 max_q
            while max_q and nums[max_q[-1]] <= val:
                max_q.pop()
            max_q.append(i - 1)
            # 维护 min_q
            while min_q and nums[min_q[-1]] >= val:
                min_q.pop()
            min_q.append(i - 1)
            
            # 调整左边界直到窗口内差值 ≤ k
            while nums[max_q[0]] - nums[min_q[0]] > k:
                if max_q[0] == left:
                    max_q.popleft()
                if min_q[0] == left:
                    min_q.popleft()
                left += 1
            
            j_min = left
            # dp[i] = sum(dp[j_min .. i-1])
            dp[i] = (prefix[i - 1] - (prefix[j_min - 1] if j_min > 0 else 0)) % MOD
            # 更新前缀和
            prefix[i] = (prefix[i - 1] + dp[i]) % MOD
        return dp[n] % MOD
```

# [2463. 最小移动总距离](https://leetcode.cn/problems/minimum-total-distance-traveled/)
X 轴上有一些机器人和工厂。给你一个整数数组 `robot` ，其中 `robot[i]` 是第 `i` 个机器人的位置。再给你一个二维整数数组 `factory` ，其中 `factory[j] = [positionj, limitj]` ，表示第 `j` 个工厂的位置在 `positionj` ，且第 `j` 个工厂最多可以修理 `limitj` 个机器人。

每个机器人所在的位置 **互不相同** 。每个工厂所在的位置也 **互不相同** 。注意一个机器人可能一开始跟一个工厂在 **相同的位置** 。

所有机器人一开始都是坏的，他们会沿着设定的方向一直移动。设定的方向要么是 X 轴的正方向，要么是 X 轴的负方向。当一个机器人经过一个没达到上限的工厂时，这个工厂会维修这个机器人，且机器人停止移动。

**任何时刻**，你都可以设置 **部分** 机器人的移动方向。你的目标是最小化所有机器人总的移动距离。

请你返回所有机器人移动的最小总距离。测试数据保证所有机器人都可以被维修。

**注意：**

- 所有机器人移动速度相同。
- 如果两个机器人移动方向相同，它们永远不会碰撞。
- 如果两个机器人迎面相遇，它们也不会碰撞，它们彼此之间会擦肩而过。
- 如果一个机器人经过了一个已经达到上限的工厂，机器人会当作工厂不存在，继续移动。
- 机器人从位置 `x` 到位置 `y` 的移动距离为 `|y - x|` 。

> **同一个工厂修理的机器人，在排序后的机器人序列里是连续的一段**。
```python
class Solution:
    def minimumTotalDistance(self, robot: List[int], factory: List[List[int]]) -> int:
        n = len(robot)
        m = len(factory)
        factory.sort(key=lambda x: x[0])
        robot.sort()
        
        @cache
        def dfs(i: int, j: int) -> int:
            # factory 在[0,i]区间修理机器人下标区间在[0,j] 机器人移动的最小总距离
            # 枚举工厂i修理k = 0,1,2...,min(j+1, limit_i)个机器人
            # 子问题是
            # 1. 修理k个机器人, dfs(i-1, j-k) + 机器人[j-k+1,j]移动到工厂i的总距离
            # 2. 不修任何机器人, dfs(i-1,j)
            
            # 边界 j<0: 没有机器人可以修理 return 0
            if j < 0:
                return 0
            # 边界 i<0: 没有工厂可以修理 return inf
            if i < 0:
                return inf
            
            # 工厂i不修理机器人
            res = dfs(i-1, j)

            # 工厂i修理k = 0,1,2...,min(j+1, limit_i)个机器人
            pos, limit = factory[i]
            dis_sum = 0
            # 枚举修理k个机器人
            for k in range(1, min(j+1, limit)+1):
                dis_sum += abs(robot[j-k+1] - pos)
                res = min(res, dfs(i-1, j-k) + dis_sum)
            return res
        return dfs(m-1, n-1)
```
