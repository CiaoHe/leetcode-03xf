# [3488. 距离最小相等元素查询](https://leetcode.cn/problems/closest-equal-element-queries/)
给你一个 **环形** 数组 `nums` 和一个数组 `queries` 。

对于每个查询 `i` ，你需要找到以下内容：

- 数组 `nums` 中下标 `queries[i]` 处的元素与 **任意** 其他下标 `j`（满足 `nums[j] == nums[queries[i]]`）之间的 **最小** 距离。如果不存在这样的下标 `j`，则该查询的结果为 `-1` 。

返回一个数组 `answer`，其大小与 `queries` 相同，其中 `answer[i]` 表示查询`i`的结果。

```python
class Solution:
    def solveQueries(self, nums: List[int], queries: List[int]) -> List[int]:
        n = len(nums)
        m = len(queries)
        res = [0] * m
        d = defaultdict(list)
        for i, num in enumerate(nums):
            d[num].append(i)

        for i, q_idx in enumerate(queries):
            target_val = nums[q_idx]
            arr = d[target_val]
            k = len(arr)
            if k == 1:
                res[i] = -1
                continue

            # 二分找到第一个>=q_idx的数
            pos = bisect_left(arr, q_idx)
            left_idx = arr[pos-1] if pos > 0 else arr[-1]
            right_idx = arr[pos+1] if pos < k-1 else arr[0]
            
            dist_left = abs(q_idx - left_idx)
            dist_left = min(dist_left, n-dist_left)
            dist_right = abs(q_idx - right_idx)
            dist_right = min(dist_right, n-dist_right)
            
            res[i] = min(dist_left, dist_right)
        return res
            
```
# [3761. 镜像对之间最小绝对距离](https://leetcode.cn/problems/minimum-absolute-distance-between-mirror-pairs/)
给你一个整数数组 `nums`。
**镜像对** 是指一对满足下述条件的下标 `(i, j)`：
- `0 <= i < j < nums.length`，并且
- `reverse(nums[i]) == nums[j]`，其中 `reverse(x)` 表示将整数 `x` 的数字反转后形成的整数。反转后会忽略前导零，例如 `reverse(120) = 21`。

返回任意镜像对的下标之间的 **最小绝对距离**。下标 `i` 和 `j` 之间的绝对距离为 `abs(i - j)`。

如果不存在镜像对，返回 `-1`。
```python
class Solution:
    def minMirrorPairDistance(self, nums: List[int]) -> int:
        def reverse(x):
            return int(str(x)[::-1])
        pos = defaultdict(list)
        for i, x in enumerate(nums):
            pos[x].append(i)
        # sort idxs for each x
        for idxs in pos.values():
            idxs.sort()
        ans = inf
        for x, idxs in pos.items():
            y = reverse(x)
            if y not in pos:
                continue
            idxs_y = pos[y]
            # 二分
            for i in idxs:
                j = bisect_right(idxs_y, i)
                if j < len(idxs_y):
                    ans = min(ans, idxs_y[j] - i)
        return ans if ans != inf else -1
```
