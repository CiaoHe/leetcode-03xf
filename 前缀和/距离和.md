# [2615. 等值距离和](https://leetcode.cn/problems/sum-of-distances)
给你一个下标从 **0** 开始的整数数组 `nums` 。现有一个长度等于 `nums.length` 的数组 `arr` 。对于满足 `nums[j] == nums[i]` 且 `j != i` 的所有 `j` ，`arr[i]` 等于所有 `|i - j|` 之和。如果不存在这样的 `j` ，则令 `arr[i]` 等于 `0` 。

返回数组 `arr` _。_


假设有值相同的下标数组：`[a₀, a₁, a₂, ..., aₘ₋₁]`
其中 `aᵢ`是原数组中的索引，且 `a₀ < a₁ < a₂ < ...`（已排序）
我们要计算对于下标 `aₖ`，到**组内所有其他元素**的距离和，即：
$sum=∣ak​−a0​∣+∣ak​−a1​∣+⋯+∣ak​−ak−1​∣+∣ak​−ak+1​∣+⋯+∣ak​−am−1​$∣

这个可以分成两部分：
- 与左边元素的距离和
- 与右边元素的距离和
因为 `a₀ < a₁ < ... < aₖ`是递增的，所以：
1. 对左边部分（j < k）：`|aₖ - aⱼ| = aₖ - aⱼ`
2. 对右边部分（j > k）：`|aₖ - aⱼ| = aⱼ - aₖ`

左边距离和的计算
- 左边有 `k`个数：`a₀, a₁, ..., aₖ₋₁`
- 左边距离和 = (ak​−a0​)+(ak​−a1​)+⋯+(ak​−ak−1​) =k⋅ak​−(a0​+a1​+⋯+ak−1​)
这里
- `k`是左边元素的个数
- `(a₀ + a₁ + ... + aₖ₋₁)`就是前 k 个元素的和，也就是 `prefix[k]`（如果 prefix[i] 是前 i 个元素的和）
所以：left_sum=k⋅ak​−prefix[k]

右边距离和的计算
- 右边有 `m - k - 1`个数：`aₖ₊₁, aₖ₊₂, ..., aₘ₋₁`
- 右边距离和 = (ak+1​−ak​)+(ak+2​−ak​)+⋯+(am−1​−ak​)=(ak+1​+ak+2​+⋯+am−1​)−(m−k−1)⋅ak​
这里：
- `(aₖ₊₁ + ... + aₘ₋₁)`就是从第 k+1 个元素到最后一个元素的和
- 可以通过前缀和计算：`prefix[m] - prefix[k+1]`
所以：
right_sum=(prefix[m]−prefix[k+1])−(m−k−1)⋅ak​

```python
class Solution:
    def distance(self, nums: List[int]) -> List[int]:
        n = len(nums)
        res = [0] * n
        d = defaultdict(list)
        for i, x in enumerate(nums):
            d[x].append(i)
        for x, idxs in d.items():
            m = len(idxs)
            if m == 1:
                continue
            
            # 计算前缀和
            prefix = [0] * (m+1)
            for i in range(m):
                prefix[i+1] = prefix[i] + idxs[i]
            
            for k, idx in enumerate(idxs):
                left_sum = k*idx - prefix[k]
                right_sum = (prefix[m] - prefix[k+1]) - (m-k-1)*idx
                res[idx] = left_sum + right_sum
        return res
```
