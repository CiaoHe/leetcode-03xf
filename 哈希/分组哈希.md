# [3741. 三个相等元素之间的最小距离 II](https://leetcode.cn/problems/minimum-distance-between-three-equal-elements-ii/)
给你一个整数数组 `nums`。
如果满足 `nums[i] == nums[j] == nums[k]`，且 `(i, j, k)` 是 3 个 **不同** 下标，那么三元组 `(i, j, k)` 被称为 **有效三元组** 。
**有效三元组** 的 **距离** 被定义为 `abs(i - j) + abs(j - k) + abs(k - i)`，其中 `abs(x)` 表示 `x` 的 **绝对值** 。
返回一个整数，表示 **有效三元组** 的 **最小** 可能距离。如果不存在 **有效三元组** ，返回 `-1`。

```python
class Solution:
    def minimumDistance(self, nums: List[int]) -> int:
        # 找到最左和最右的下标i k 计算最小的2(k-i)
        ans = inf
        n = len(nums)
        pos = defaultdict(list)
        for i, num in enumerate(nums):
            pos[num].append(i)
        # sort pos list by index
        for num in pos:
            pos[num].sort()
        # find the minimum distance
        for num in pos:
            if len(pos[num]) < 2:
                continue
            p = pos[num]
            for i in range(2, len(p)):
                ans = min(ans, 2*(p[i]-p[i-2]))
        return ans if ans != inf else -1
```

# [3121. 统计特殊字母的数量 II](https://leetcode.cn/problems/count-the-number-of-special-characters-ii/)
给你一个字符串 `word`。如果 `word` 中同时出现某个字母 `c` 的小写形式和大写形式，并且 **每个** 小写形式的 `c` 都出现在第一个大写形式的 `c` 之前，则称字母 `c` 是一个 **特殊字母** 。

返回 `word` 中 **特殊字母** 的数量。

```python
class Solution:
    def numberOfSpecialChars(self, word: str) -> int:
        cnt = 0
        last_lower = {}
        first_upper = {}
        for i,c in enumerate(word):
            if c.islower():
                last_lower[c] = i
            else:
                # 只有第一次出现upper时候才记录
                if c not in first_upper:
                    first_upper[c] = i
        
        cnt = 0
        for c in last_lower:
            upper_c = c.upper()
            if upper_c in first_upper and last_lower[c] < first_upper[upper_c]:
                cnt += 1
        return cnt
```
