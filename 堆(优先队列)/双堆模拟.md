# [2402. 会议室 III](https://leetcode.cn/problems/meeting-rooms-iii/)
难度分 2092[第309场周赛Q4](https://leetcode.cn/contest/weekly-contest-309 "访问LeetCode竞赛链接")等级 7

给你一个整数 `n` ，共有编号从 `0` 到 `n - 1` 的 `n` 个会议室。

给你一个二维整数数组 `meetings` ，其中 `meetings[i] = [starti, endi]` 表示一场会议将会在 **半闭** 时间区间 `[starti, endi)` 举办。所有 `starti` 的值 **互不相同** 。

会议将会按以下方式分配给会议室：

1. 每场会议都会在未占用且编号 **最小** 的会议室举办。
2. 如果没有可用的会议室，会议将会延期，直到存在空闲的会议室。延期会议的持续时间和原会议持续时间 **相同** 。
3. 当会议室处于未占用状态时，将会优先提供给原 **开始** 时间更早的会议。

返回举办最多次会议的房间 **编号** 。如果存在多个房间满足此条件，则返回编号 **最小** 的房间。

**半闭区间** `[a, b)` 是 `a` 和 `b` 之间的区间，**包括** `a` 但 **不包括** `b` 。

```python
class Solution:
    def mostBooked(self, n: int, meetings: List[List[int]]) -> int:        
        # meetings is sorted by start time
        meetings.sort(key=lambda x: x[0])
        
        # first heap: track the num of all the rooms that are free
        free_rooms = [i for i in range(n)]
        heapify(free_rooms)

        # second heap: track the end times of all the meetings that are currently booked and the room number
        # (end_time, room_number)
        busy_rooms = []
        
        meetings_count = [0] * n

        for start, end in meetings:
            duration = end - start
            # 释放所有已经结束的房间
            while busy_rooms and busy_rooms[0][0] <= start:
                _, room = heappop(busy_rooms)
                heappush(free_rooms, room)
                
            # 安排会议
            if free_rooms:
                room = heappop(free_rooms)
                heappush(busy_rooms, (end, room))
                meetings_count[room] += 1
            else:
                # 等待最早结束的会议
                earliest_end_time, room = heappop(busy_rooms)
                heappush(busy_rooms, (earliest_end_time + duration, room))
                meetings_count[room] += 1
        return meetings_count.index(max(meetings_count))
```
