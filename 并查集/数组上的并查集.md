# [2612. 最少翻转操作数](https://leetcode.cn/problems/minimum-reverse-operations)
难度分 2824

给定一个整数 `n` 和一个整数 `p`，它们表示一个长度为 `n` 且除了下标为 `p` 处是 `1` 以外，其他所有数都是 `0` 的数组 `arr`。同时给定一个整数数组 `banned` ，它包含数组中的一些限制位置。在 `arr` 上进行下列操作：

- 如果单个 1 不在 `banned` 中的位置上，反转大小为 `k` 的 **子数组**。

返回一个包含 `n` 个结果的整数数组 `answer`，其中第 `i` 个结果是将 `1` 放到位置 `i` 处所需的 **最少** 翻转操作次数，如果无法放到位置 `i` 处，此数为 `-1` 。

> 1. 把0到n-1看成n个节点，设置i反转后的位置为j，在i和j之间连边，边权重1
> 	1. 问题变成：计算p到各个点的最短路径 -> 可以用BFS计算
> 2. 如果反转转子数组 \[L,R\] 
> 	1. 观察到
> 		1. 下标 L 翻转到下标 R。
> 		2. 下标 L+1 翻转到下标 R−1。
> 		3. 下标 L+2 翻转到下标 R−2。
> 		4. 依此类推，可以发现翻转前后的下标之和恒等于 L+R，所以下标 i 在翻转后的下标是 L+R−i。
> 	2. 当子数组向右滑动时，每滑动一个单位，L 和 R 都增加 1，所以翻转后的下标会增加 2。
> 	3. 当子数组向左滑动时，每滑动一个单位，L 和 R 都减少 1，所以翻转后的下标会减少 2。
> 	4. 因此，**i 翻转后的下标组成了一个公差为 2 的等差数列**。
> 3. i 翻转后的下标，
> 	1. 最小：
> 		1. 当 i 在子数组右端点时，可以翻转到子数组左端点 i−k+1，这是最小值。
> 			1. i<k−1 的情况，当子数组在最左边时，L=0, R=k−1，i 翻转后是 L+R−i=0+(k−1)−i=k−i−1，小于 k−i−1 的下标无法翻转得到。
> 		2. 因此 i 翻转后的**最小值**为 $max(i−k+1,k−i−1)$。
> 	2. 最大：
> 		1. 当 i 在子数组左端点时，可以翻转到子数组右端点 i+k−1，这是最大值
> 			1. i>n−k 的情况，当子数组在最右边时，L=n−k, R=n−1，i 翻转后是 L+R−i=(n−k)+(n−1)−i=2n−k−i−1，大于 2n−k−i−1 的下标无法翻转到。
> 		2. i 翻转后的**最大值**为 $min(i+k−1,2n−k−i−1)$

> 由于等差数列的公差为 2，翻转后的下标要么都是偶数，要么都是奇数。我们用两个有序集合 $indices_0$ 和 $indices_1$ 来维护**没有访问过的**偶数下标和奇数下标。注意这些下标不能在 banned 中。由于 p 是起点（已访问），所以也不在有序集合中。
```python
class Solution:
    def minReverseOperations(self, n: int, p: int, banned: List[int], k: int) -> List[int]:
        ban = set(banned) | {p}
        indices = [SortedList(), SortedList()] # (even, odd)
        for i in range(n):
            if i not in ban:
                indices[i%2].add(i)
        indices[0].add(n)
        indices[1].add(n)
        
        ans = [-1] * n
        ans[p] = 0 # 起点
        q = deque([p])
        while q:
            i = q.popleft()
            # indices[mn % 2]中从 mn 到 mx 的所有下标都可以由i翻转得到
            mn = max(i-k+1, k-i-1)
            mx = min(i+k-1, 2*n-k-i-1)
            access = indices[mn % 2]
            idx = access.bisect_left(mn)
            while access[idx] <= mx:
                j = access.pop(idx)
                ans[j] = ans[i] + 1
                q.append(j)
        return ans
```

# [3666. 使二进制字符串全为 1 的最少操作次数](https://leetcode.cn/problems/minimum-operations-to-equalize-binary-string/)

给你一个二进制字符串 `s` 和一个整数 `k`。
在一次操作中，你必须选择 **恰好** `k` 个 **不同的** 下标，并将每个 `'0'` **翻转** 为 `'1'`，每个 `'1'` 翻转为 `'0'`。
返回使字符串中所有字符都等于 `'1'` 所需的 **最少** 操作次数。如果不可能，则返回 -1。

参考[[[2612. 最少翻转操作数](https://leetcode.cn/problems/minimum-reverse-operations)](#[2612.%20最少翻转操作数](https%20//leetcode.cn/problems/minimum-reverse-operations))]
```python
class Solution:
    def minOperations(self, s: str, k: int) -> int:
        n = len(s)
        not_vis = [SortedList(range(0,n+1,2)), SortedList(range(1,n+1,2))]
        # 维护没有访问过的偶数和奇数的位置
        not_vis[0].add(n+1)
        not_vis[1].add(n+1)

        ans = 0
        # z: 0的个数
        z_start = s.count('0')
        not_vis[z_start % 2].discard(z_start)
        q = [z_start]

        # bfs
        while q:
            tmp = q
            q = []
            for z in tmp:
                if z == 0:
                    return ans
                # 假设翻转x个0
                # 那么z' = z - x + (k - x) = z + k - 2x
                # 但是x的上限在 min(k, z) 之间
                # x的下限在 max(0, k - n + z) 之间
                mn_z_next = z + k - 2 * min(k, z)
                mx_z_next = z + k - 2 * max(0, k - n + z)
                sl = not_vis[mn_z_next % 2]
                idx = sl.bisect_left(mn_z_next)
                while sl[idx] <= mx_z_next:
                    # 都要删走
                    j = sl.pop(idx)
                    q.append(j)
            ans += 1
        return -1
```
