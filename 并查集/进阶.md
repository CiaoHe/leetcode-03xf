# [3600. 升级后最大生成树稳定性](https://leetcode.cn/problems/maximize-spanning-tree-stability-with-upgrades/)

给你一个整数 `n`，表示编号从 0 到 `n - 1` 的 `n` 个节点，以及一个 `edges` 列表，其中 `edges[i] = [ui, vi, si, musti]`：

- `ui` 和 `vi` 表示节点 `ui` 和 `vi` 之间的一条无向边。
- `si` 是该边的强度。
- `musti` 是一个整数（0 或 1）。如果 `musti == 1`，则该边 **必须** 包含在生成树中，且 **不能****升级** 。

你还有一个整数 `k`，表示你可以执行的最多 **升级** 次数。每次升级会使边的强度 **翻倍** ，且每条可升级边（即 `musti == 0`）最多只能升级一次。

一个生成树的 **稳定性** 定义为其中所有边的 **最小** 强度。

返回任何有效生成树可能达到的 **最大** 稳定性。如果无法连接所有节点，返回 `-1`。

**注意：** 图的一个 **生成树**（**spanning tree**）是该图中边的一个子集，它满足以下条件：

- 将所有节点连接在一起（即图是 **连通的** ）。
- **不** 形成任何环。
- 包含 **恰好** `n - 1` 条边，其中 `n` 是图中节点的数量。

> **最大化最小值** + **并查集** + **最小生成树**
```python
class UnionFind:
    def __init__(self, n):
        self.fa = {}
        self.cnt = 0
        for i in range(n):
            self.add(i)
    
    def add(self, x):
        if x not in self.fa:
            self.fa[x] = x
            self.cnt += 1
    
    def merge(self, x, y):
        x = self.find(x)
        y = self.find(y)
        if x != y:
            self.fa[x] = y
            self.cnt -= 1
    
    def find(self, x):
        self.add(x)
        if self.fa[x] != x:
            self.fa[x] = self.find(self.fa[x])
        return self.fa[x]

    def connected(self, x, y):
        return self.find(x) == self.find(y)
    
class Solution:
    def maxStability(self, n: int, edges: List[List[int]], k: int) -> int:
        def check(limit, k, n):
            """
            假设我已经定好了一个目标及格线 limit, 我能不能在这个及格线之上 拼出一棵完整的生成树
            """
            uf = UnionFind(n)
            heap = []
            for x,y,s,f in edges:
                if f == 1: # 必须作为边
                    if s < limit: # 强度不够
                        # 如果原始强度连当前的及格线 limit 都达不到 直接false
                        return False
                    # 如果存在环 也g
                    if uf.connected(x, y):
                        return False
                    uf.merge(x, y)
                else:
                    # 剩下的边都是我们可以选择性使用的, 把所有备胎边丢进一个最大堆里面
                    # 是否需要升级 先放入最大堆
                    heappush(heap, (-s, x, y))
            
            # 贪心组装 (最大堆出堆)
            # 只要图还没完全连通 (uf.cnt != 1) 而且 heap还有备料
            while uf.cnt != 1 and heap:
                s, x, y = heappop(heap)
                s = -s
                # 如果这边的两头已经连过了，这条边就是废边，直接扔掉
                if uf.connected(x, y):
                    continue
                # 白嫖优先,这条边本身就很强，直接达到及格线。太好了，不用花升级次数，直接连上！
                if s >= limit:
                    uf.merge(x, y)
                # 氪金升级
                elif s * 2 >= limit and k>0:
                    uf.merge(x, y)
                    k -= 1
                # 及时止损
                # 这是最大堆最精妙的地方。
                # 因为我们是按强度从大到小拿的，如果连当前这条最强的边，就算翻倍了也够不到及格线
                else:
                    break
            return uf.cnt == 1
        
        left, right = 0, 10**9
        while left < right:
            mid = (left + right) // 2
            if check(mid, k, n):
                left = mid + 1
            else:
                right = mid
        return left - 1
```
