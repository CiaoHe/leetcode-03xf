# [1536. 排布二进制网格的最少交换次数](https://leetcode.cn/problems/minimum-swaps-to-arrange-a-binary-grid/)
给你一个 `n x n` 的二进制网格 `grid`，每一次操作中，你可以选择网格的 **相邻两行** 进行交换。

一个符合要求的网格需要满足主对角线以上的格子全部都是 **0** 。

请你返回使网格满足要求的最少操作次数，如果无法使网格符合要求，请你返回 **-1** 。

主对角线指的是从 `(1, 1)` 到 `(n, n)` 的这些格子。

```python
class Solution:
    def minSwaps(self, grid: List[List[int]]) -> int:
        n = len(grid)
        # 统计每一行最右边连续0的个数
        zeros = [0] * n
        for i in range(n):
            tmp = 0
            for j in range(n-1, -1, -1):
                if grid[i][j] == 0:
                    tmp += 1
                else:
                    break
            zeros[i] = tmp
        
        ans = 0
        for i in range(n-1):
            target = n-i-1 # 至少需要n-i-1个0
            found = -1

            for j in range(i, n):
                if zeros[j] >= target:
                    found = j
                    break
            if found == -1:
                return -1
            ans += found - i # 对调轮数
            val = zeros.pop(found)
            zeros.insert(i, val)
        return ans
```
