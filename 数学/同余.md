# [2598. 执行操作后的最大 MEX](https://leetcode.cn/problems/smallest-missing-non-negative-integer-after-operations/)
给你一个下标从 0 开始的整数数组 nums 和一个整数 value 。

在一步操作中，你可以对 nums 中的任一元素加上或减去 value 。

例如，如果 nums = [1,2,3] 且 value = 2 ，你可以选择 nums[0] 减去 value ，得到 nums = [-1,2,3] 。
数组的 MEX (minimum excluded) 是指其中数组中缺失的最小非负整数。

例如，[-1,2,3] 的 MEX 是 0 ，而 [1,0,3] 的 MEX 是 2 。
返回在执行上述操作 任意次 后，nums 的最大 MEX 。


```python
class Solution:
    def findSmallestInteger(self, nums: List[int], value: int) -> int:
        n = len(nums)
        # 算出来所有的x%value的值，然后排序，然后从0开始，找到第一个没有出现的数
        mod_nums = [x%value for x in nums]
        freq = Counter(mod_nums)
        i = 0
        while True:
            mod = i % value # 这是我们要找的能够经过若干次操作得到的target值
            if freq[mod] > 0:
                freq[mod] -= 1
                i += 1
            else:
                return i
```
