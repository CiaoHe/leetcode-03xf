# [396. 旋转函数](https://leetcode.cn/problems/rotate-function/)
给定一个长度为 `n` 的整数数组 `nums` 。

假设 `arrk` 是数组 `nums` 顺时针旋转 `k` 个位置后的数组，我们定义 `nums` 的 **旋转函数**  `F` 为：

- `F(k) = 0 * arrk[0] + 1 * arrk[1] + ... + (n - 1) * arrk[n - 1]`

返回 _`F(0), F(1), ..., F(n-1)`中的最大值_ 。

生成的测试用例让答案符合 **32 位** 整数

```python
class Solution:
    def maxRotateFunction(self, nums: List[int]) -> int:
        n = len(nums)
        s = sum(nums)
        f = sum(i * nums[i] for i in range(n))
        ans = f
        for i in range(n-1, 0, -1):
            # 原来的f(i) = 0 * nums[0] + 1 * nums[1] + ... + (n-1) * nums[n-1]
            # 旋转一次之后，新的f(i)' = 0 * nums[n-1] + 1 * nums[0] + 2 * nums[1] + ... + (n-1) * nums[n-2]
            # f(i)' - f(i) = nums[0] + ... + nums[n-2] - (n-1) * nums[n-1] = s - n * nums[i]
            f += s - n * nums[i]
            ans = max(ans, f)
        return ans
```
