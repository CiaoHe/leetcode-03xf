# [1963. 使字符串平衡的最小交换次数](https://leetcode.cn/problems/minimum-number-of-swaps-to-make-the-string-balanced/)
贪心写法
```python
class Solution:
    def minSwaps(self, s: str) -> int:
        n = len(s)
        s = list(s)
        # 如果遇到'[', +1, 如果遇到']', -1
        ans = 0
        # 从左到右统计没有匹配的'['的个数
        # 如果遇到[, c+=1, 如果遇到], c-=1, 如果c<0, 则需要交换
        c = 0
        n = len(s)
        j = n-1
        for i, ch in enumerate(s):
            if ch == '[':
                c += 1
            else:# ch == ']'
                if c > 0:
                    c -= 1
                else:
                    # 需要找到最右边的[, 进行交换
                    while j >= 0 and s[j] != '[':
                        j-=1
                    # 交换
                    s[i],s[j] = s[j], s[i]
                    ans += 1
                    c += 1
        return ans

```
# [2116. 判断一个括号字符串是否有效](https://leetcode.cn/problems/check-if-a-parentheses-string-can-be-valid/)
```python
class Solution:
    def canBeValid(self, s: str, locked: str) -> bool:
        n = len(s)
        if n % 2 == 1:
            return False
        # c: 当前未匹配的左括号数量, left->right, 遇到左括号+1, 遇到右括号-1
        # 如果任何时候c>=0 且结尾的时候c=0, 则返回True
        
        # 我们维护一个c的set, 记录所有可能的c的值
        # 更进一步我们发现，c_set内部是连续的奇数/偶数，因此我们只需要维护mn, mx
        mn, mx = 0, 0
        # 如果locked_i = 1, 那么不可以修改; 如果s_i = (, 那么mn, mx = mn+1, mx+1; 如果s_i = ), 那么mn, mx = mn-1, mx-1; 此时如果mx<0, 则返回False； 如果mn<0, 则更新mn为最小非负数
        # 如果locked_i = 0, 那么可以修改为左括号，即mx+=1, 也可以修改为右括号，即mn-=1; 如果locked_i = 0. 如果mn=-1, 则把mn更新为1

        for si, lock in zip(s, locked):
            if lock == '1': # 不能修改
                if si == '(':
                    mn += 1
                    mx += 1
                elif si == ')':
                    mx -= 1
                    if mx < 0:
                        return False
                    mn -= 1
            else: # 可以修改
                mx += 1 # 修改为左括号
                mn -= 1 # 修改为右括号
            if mn < 0:
                mn = 1
        return mn == 0
                
```
更通俗的写法，两次遍历
1. 从左到右尝试把所有的 右括号 都更换为 左括号
2. 从右到左常侍把所有的 左括号 都更换为 右括号
```python
class Solution:
    def canBeValid(self, s: str, locked: str) -> bool:
        n = len(s)
        if n % 2 == 1:
            return False
        
        c = 0
        for si, lock in zip(s, locked):
            if si == '(' or lock == '0':
                c += 1
            elif c:
                c -= 1
            else:
                return False
        
        c = 0
        for si, lock in zip(s[::-1], locked[::-1]):
            if si == ')' or lock == '0':
                c += 1
            elif c:
                c -= 1
            else:
                return False
        return True
```

# [761. 特殊的二进制字符串](https://leetcode.cn/problems/special-binary-string/)
难度分: 2292

**特殊的二进制字符串** 是具有以下两个性质的二进制序列：

- `0` 的数量与 `1` 的数量相等。
- 二进制序列的每一个前缀码中 `1` 的数量要大于等于 `0` 的数量。

给定一个特殊的二进制字符串 `s`。

一次移动操作包括选择字符串 `s` 中的两个连续的、非空的、特殊子串，并交换它们。两个字符串是连续的，如果第一个字符串的最后一个字符与第二个字符串的第一个字符的索引相差正好为 1。

返回在字符串上应用任意次操作后可能得到的字典序最大的字符串。

```python
class Solution:
    def makeLargestSpecial(self, s: str) -> str:
        n = len(s)
        if n <= 2:
            return s
        
        # 把s划分成若干段合法的字符串 记录在substring里面
        substring = []
        diff = 0 # 1的个数 - 0的个数
        start = 0 # 当前子串的起始index
        for i, ch in enumerate(s):
            if ch == '1':
                diff += 1
            else:
                diff -= 1
                # 注意这里必须在0的个数追上1的个数时再处理balance的情况
                # 因为如果0的个数多于1的个数 此时虽然追加1 但是不符合平衡的定义
                if diff == 0:
                    # 此时[start, i] 这段是合法的平衡子串 且没有办法继续划分，只能考虑嵌套
                    substring.append('1' + self.makeLargestSpecial(s[start+1:i]) + '0')
                    start = i + 1
        
        substring.sort(reverse=True)
        return ''.join(substring)
```
